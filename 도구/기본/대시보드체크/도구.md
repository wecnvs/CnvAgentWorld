# 대시보드체크

대시보드 프론트(`시스템/대시보드/웹`의 js·css·html)나 서버를 고친 뒤 **완료 보고 전** 실행하는 회귀 체크 러너다.

## 실행
```
sh 도구/기본/대시보드체크/체크.sh
```

## 검사 항목
1. 웹 JS 전체 `node --check` 문법.
2. `test_dashboard_static.py` — 캐시버전 `?v=` 단일 일관성·필수 구조·API 경로.
3. `test_dashboard_mobile_playwright.py`(모바일 iOS viewport·has_touch) — 공간 진입·**콤보박스 공간전환 렌더+터치**·스플릿바 탭 후 터치 프리즈 없음·입력 탭·스크롤·전송 정렬.

통과(exit 0)해야 완료다. 정본 체크리스트: `관리기록/대시보드_수정_체크리스트.md`. 규칙: `law_manager.md`「모바일 터치·인터랙션 회귀」. headless가 못 잡는 iOS 특이 버그는 실기기로 확인.
