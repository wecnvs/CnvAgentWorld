# cu-mac (도구 묶음)

macOS 컴퓨터유즈를 위한 **원자 기능 16종**. 스킬 `computer-use-mac`이 이 도구들을 조합해 호출한다(스킬은 방법, 이 폴더는 부품). 실제 조작 절차·안전 규칙·실전 케이스는 스킬 쪽(`SKILL.md`/`cases.jsonl`)을 따르고, 여기서는 도구의 호출 인터페이스만 정의한다.

## 실행 래퍼 (항상 이걸로 호출)
```bash
bash 도구/기본/cu-mac/run_tool.sh <script.py> [인수...]
```
- `run_tool.sh`는 자기 폴더(`BASH_SOURCE` dir) 기준으로 python·라이브러리 경로를 잡아 실행한다 → 폴더 위치가 바뀌어도 동작.
- macOS 전용: `/opt/homebrew/bin/python3.13` + expat 라이브러리 경로를 자동 설정(머신 환경 의존, 비밀값 아님).

## 도구 목록 (16)
| 도구 | 한 일 | 호출 예 |
|---|---|---|
| `run_tool.sh` | 파이썬 도구 실행 래퍼(환경 자동처리) | `bash run_tool.sh screenshot.py /tmp/cu_screen.png` |
| `screenshot.py` | 화면 캡처(포인터 위치 빨강 마커 오버레이, HUD 제외) | `run_tool.sh screenshot.py /tmp/cu_screen.png` |
| `move.py` | 포인터 이동만(클릭 안 함) | `run_tool.sh move.py <x> <y>` |
| `click.py` | 클릭(좌/우/더블) | `run_tool.sh click.py <x> <y> [--right|--double]` |
| `click_flash.py` | 클릭 + 위치 시각 플래시 | `run_tool.sh click_flash.py <x> <y>` |
| `type_text.py` | 텍스트 입력(한국어 포함) | `run_tool.sh type_text.py "텍스트"` |
| `press_key.py` | 키/단축키 입력 | `run_tool.sh press_key.py "command+space"` |
| `scroll.py` | 스크롤 | `run_tool.sh scroll.py <x> <y> down [n]` |
| `find_element.py` | 창/주소창 등 좌표 조회(접근성) | `run_tool.sh find_element.py windows [App]` |
| `web_locate.py` | 브라우저 요소 정확좌표(CDP getBoundingClientRect) | `run_tool.sh web_locate.py "버튼 텍스트"` |
| `log.py` | HUD에 상태/판단 로그(`--think`/`--done`) | `run_tool.sh log.py "메시지"` |
| `overlay.py` | HUD 오버레이 앱 본체(스크린샷에 안 찍힘) | (overlay_start.sh가 기동) |
| `overlay_start.sh` | HUD 오버레이 시작/중지/상태 | `bash overlay_start.sh [start|stop|status]` |
| `ansi_filter.py` | 터미널 ANSI 코드 제거 필터 | (launch_cu.sh가 사용) |
| `launch_cu.sh` | 에이전트 CLI 실행 + 터미널 전체 HUD 스트리밍(Claude Code용) | `bash launch_cu.sh` |
| `cu_lock.py` | 컴퓨터유즈 동시조작 방지 락(획득/연장/해제/상태) | `run_tool.sh cu_lock.py acquire <agent_id> --note "작업"` |

## 안전·보안 (스킬에서 강제)
- 동시 화면충돌 방지: GUI 조작 전 `cu_lock.py acquire`, 긴 작업 중 `heartbeat`, 끝나면 반드시 `release`.
- 안전 클릭: 추측 클릭 금지 — `move → screenshot(포인터 마커 확인) → click`.
- 비밀값(비밀번호·OTP·토큰·연결정보)은 인수/로그/캡처 어디에도 평문으로 남기지 않는다(law.md §7 보안 불변식). 민감 화면 캡처는 공개 경로에 두지 않는다.

> 호출 절차·실패 분기·실전 교훈은 스킬 `computer-use-mac`을 따른다. 컴퓨터유즈 공통 원칙은 `computer-use-charter` 헌장을 먼저 적용한다(발견기로 찾아 적용).
