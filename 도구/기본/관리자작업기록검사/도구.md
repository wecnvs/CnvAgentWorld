# 관리자작업기록검사

관리자에이전트 작업 후 `관리기록/` 정본 폴더와 `law_manager.md` 기록 의무가 유지되는지 검사한다.

```sh
python3 도구/기본/관리자작업기록검사/검사.py
python3 도구/기본/관리자작업기록검사/검사.py --strict-mtime
```

`--strict-mtime`은 시스템·대시보드·지침·도구 같은 감시 대상 파일이 최신 작업 로그보다 나중에 수정됐는지 확인한다. 훅이나 최종 검증에서 이 옵션을 사용해 작업이력 누락을 막는다.

수명주기 훅은 공통 래퍼를 호출한다.

```sh
python3 도구/기본/관리자작업기록검사/훅.py --mode claude-stop
python3 도구/기본/관리자작업기록검사/훅.py --mode agy-post-invocation
python3 도구/기본/관리자작업기록검사/훅.py --mode agy-stop
```

- Claude Code: `.claude/settings.json`의 `Stop` 훅이 `.claude/hooks/manager-history-stop.sh`를 실행한다.
- agy: `.agents/hooks.json`의 `PostInvocation`·`Stop` 훅이 `.agents/hooks/manager-history-check.sh`를 실행한다.
- 훅이 실패하면 작업로그를 갱신하고 `--strict-mtime` 검사를 다시 통과시킨 뒤 완료 보고한다.
