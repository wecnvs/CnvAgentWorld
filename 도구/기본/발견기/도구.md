# 발견기 (도구)

frontmatter `description`을 스캔해 요청에 맞는 자원 후보를 추리는 실행 도구.

```
python3 도구/기본/발견기/발견.py "<찾는 내용>" [--type skill|knowledge|tool|asset|all] [--top 5]
```

- 매번 라이브 스캔 → 별도 색인·태그 불필요(드리프트 없음).
- **후보 하나가 아니라 여러 개**를 점수순으로 준다. 최종 선택·활용은 부르는 에이전트가 후보 description을 보고 판단.
- 점수는 값싼 1차 추림(검색어 겹침)일 뿐 — 정밀 매칭은 에이전트의 의미 판단이 한다.

언제/어떻게 쓰는지는 스킬 `스킬/기본/발견기/`를 따른다.
